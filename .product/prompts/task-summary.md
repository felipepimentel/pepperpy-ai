Generate a summary of tasks from `.product/status.md`:

1. Status Overview:
   - Count tasks by status:
     - To Do
     - In Progress
     - Done
   - Group by priority level
   - Note recent changes
   - Identify trends

2. Progress Analysis:
   - Tasks completed this week
   - Tasks started this week
   - Blocked tasks
   - Upcoming deadlines
   - Velocity metrics

3. Priority Distribution:
   - High priority tasks
   - Medium priority tasks
   - Low priority tasks
   - Priority shifts
   - Resource allocation

4. Timeline Assessment:
   - Current sprint status
   - Milestone progress
   - Delayed tasks
   - Risk factors
   - Projected completion

5. Summary Report:
   - Key achievements
   - Major blockers
   - Resource needs
   - Recommendations
   - Next steps
