  Use this prompt when:

  - Updating project goals
  
  The prompt helps maintain a clear vision of project
  direction and milestones.

Review and update the project roadmap:
1. Analyze current project status:
   - Review completed milestones
   - Check ongoing development
   - Assess pending features
   - Review technical debt

2. Update milestone planning:
   - Short-term goals (1-2 months)
   - Medium-term objectives (3-6 months)
   - Long-term vision (6+ months)

3. For each planned feature:
   - Estimate development effort
   - Identify dependencies
   - Set target completion dates
   - Define success criteria

4. Document updates in:
   - `/docs/roadmap.md`
   - `/docs/status.md` (new tasks)
   - Release planning documents
