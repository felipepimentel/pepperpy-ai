scope: "Pepperpy Project"
description: |
  A systematic guide for reviewing and adjusting task priorities.
  Use this prompt when:
  - Conducting sprint planning
  - Evaluating task urgency
  - Balancing resources
  - Updating project timelines
  - Responding to changing requirements
  
  The prompt helps maintain focus on high-impact tasks while
  ensuring critical deadlines are met.
---

Review task priorities in `/docs/status.md`:
1. Analyze tasks in each section (To Do, In Progress, Done).
2. For each task:
   - Verify current priority level (High, Medium, Low).
   - Check dependencies and blockers.
   - Review estimated completion time.
   - Consider impact on project timeline.
3. Suggest priority adjustments based on:
   - Technical dependencies
   - Resource availability
   - Project deadlines
   - Business value
4. Update task priorities and document rationale.
