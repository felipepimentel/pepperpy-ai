title: Feedback Loop
version: "1.0"
scope: "Pepperpy Project"
description: |
  A guide for collecting and incorporating task feedback.
  Use this prompt when:
  - Reviewing completed tasks
  - Gathering insights
  - Improving processes
  - Learning from experience
  - Planning improvements
  
  The prompt helps create a continuous learning cycle
  for project improvement.
---

Review recently completed tasks in `.product/status.md`:

1. Task Review:
   - List recently completed tasks
   - Review implementation details
   - Check acceptance criteria
   - Note completion time
   - Document outcomes

2. Success Analysis:
   - What worked well
   - Effective approaches
   - Helpful resources
   - Positive outcomes
   - Time savers

3. Challenge Review:
   - Encountered difficulties
   - Unexpected issues
   - Resource constraints
   - Timeline impacts
   - Technical challenges

4. Learning Points:
   - Key insights
   - Best practices
   - Process improvements
   - Technical learnings
   - Team coordination

5. Recommendations:
   - Process adjustments
   - Tool improvements
   - Documentation updates
   - Training needs
   - Future considerations

6. Documentation:
   - Update task entries
   - Add feedback notes
   - Log improvements
   - Set review dates
   - Track patterns
