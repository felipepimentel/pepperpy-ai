title: New Task Prompt
version: "1.0"
scope: "Pepperpy Project"
description: |
  A template for creating new tasks in the project's status tracking system.
  Use this prompt when:
  - Adding a new feature or enhancement
  - Creating a bug fix task
  - Planning refactoring work
  - Setting up documentation tasks
  
  The prompt ensures consistent task creation with all necessary metadata
  and proper categorization in the project's task management system.
---

Please create a new task in the `.product/kanban.md` file under the **To Do** section.  
Details to include:
- Unique task ID.
- Task title.
- Priority level (High, Medium, Low).
- Brief description of the task.
- Associated components or files.
