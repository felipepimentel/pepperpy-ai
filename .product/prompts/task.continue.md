description: |
  A guide for resuming and progressing with ongoing tasks.
  Use this prompt when:
  - Returning to an in-progress task
  - Checking task status
  - Planning next steps
  - Identifying blockers
  - Updating task progress
  
  The prompt helps maintain momentum and clear direction
  on ongoing tasks.
---

Review tasks marked as **In Progress** in `.product/kanban.md`:

1. For the selected task:
   - Review current progress and last update
   - Check completed subtasks
   - Verify remaining work
   - Identify any new dependencies

2. Plan next steps:
   - Break down remaining work
   - Set immediate objectives
   - Estimate time to completion
   - Note potential risks

3. Update task status:
   - Log progress made
   - Add new subtasks if needed
   - Update timestamp
   - Document any blockers

4. If no task is **In Progress**:
   - Review **To Do** list
   - Select highest priority task
   - Move to **In Progress**
   - Begin task breakdown
