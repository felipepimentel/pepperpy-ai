# Documentação dos Módulos PepperPy

Este documento fornece uma visão geral dos módulos disponíveis no framework PepperPy e como utilizá-los em seus projetos.

## Módulos Principais

O PepperPy é organizado em módulos que fornecem funcionalidades específicas:

- [Core](./modules/core.md) - Componentes e abstrações fundamentais
- [LLM](./modules/llm.md) - Integração com modelos de linguagem
- [Agents](./modules/agents.md) - Agentes autônomos e interativos
- [RAG](./modules/rag.md) - Retrieval Augmented Generation
- [Memory](./modules/memory.md) - Gerenciamento de memória para contexto
- [Embedding](./modules/embedding.md) - Geração e manipulação de embeddings
- [Storage](./modules/storage.md) - Armazenamento de dados
- [Workflows](./modules/workflows.md) - Orquestração de fluxos de trabalho
- [Caching](./modules/caching.md) - Mecanismos de cache
- [Analysis](./modules/analysis.md) - Análise de dados e conteúdo
- [Capabilities](./modules/capabilities.md) - Recursos e capacidades do framework
- [Evaluation](./modules/evaluation.md) - Avaliação de modelos e resultados
- [Multimodal](./modules/multimodal.md) - Processamento multimodal (visão, áudio)
- [Observability](./modules/observability.md) - Monitoramento e observabilidade
- [Security](./modules/security.md) - Segurança e controle de acesso
- [CLI](./modules/cli.md) - Interface de linha de comando
- [Cloud](./modules/cloud.md) - Integração com serviços em nuvem
- [Hub](./modules/hub.md) - Integração com PepperHub
- [Formats](./modules/formats.md) - Formatos de dados e conversões
- [Adapters](./modules/adapters.md) - Adaptadores para diferentes serviços
- [Optimization](./modules/optimization.md) - Otimização de desempenho
- [Pipeline](./modules/pipeline.md) - Pipelines de processamento
