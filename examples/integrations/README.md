# PepperPy Integrations Examples

Este diretório contém exemplos demonstrando como usar as funcionalidades de integração do framework PepperPy com serviços externos.

## Exemplos Disponíveis

### Complete Flow

O script `complete_flow.py` demonstra:

- Fluxo completo de integração
- Conexão com múltiplos serviços
- Orquestração de dados entre sistemas

Para executar:

```bash
python examples/integrations/complete_flow.py
```

### Intent to Composition

O script `intent_to_composition.py` demonstra:

- Conversão de intenções em composições
- Mapeamento de comandos para fluxos de trabalho
- Automação baseada em linguagem natural

Para executar:

```bash
python examples/integrations/intent_to_composition.py
```

### Template to Intent

O script `template_to_intent.py` demonstra:

- Uso de templates para reconhecimento de intenções
- Extração de parâmetros de comandos
- Processamento de linguagem natural estruturada

Para executar:

```bash
python examples/integrations/template_to_intent.py
```

## Requisitos

Todos os exemplos requerem que o framework PepperPy esteja instalado. Você pode instalá-lo usando:

```bash
pip install -e ../..
```

ou

```bash
poetry install
```

a partir do diretório raiz do repositório.
