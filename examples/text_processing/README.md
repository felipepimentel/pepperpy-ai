# PepperPy Text Processing Examples

Este diretório contém exemplos demonstrando como usar as funcionalidades de processamento de texto do framework PepperPy.

## Exemplos Disponíveis

### Basic Composition

O script `basic_composition.py` demonstra:

- Composição básica de processadores de texto
- Transformação e filtragem de texto
- Encadeamento de operações de processamento

Para executar:

```bash
python examples/text_processing/basic_composition.py
```

### Document Summarizer

O script `document_summarizer.py` demonstra:

- Sumarização de documentos longos
- Extração de informações-chave
- Geração de resumos estruturados

Para executar:

```bash
python examples/text_processing/document_summarizer.py
```

### Multilingual Translator

O script `multilingual_translator.py` demonstra:

- Tradução entre múltiplos idiomas
- Detecção automática de idioma
- Preservação de formatação e contexto

Para executar:

```bash
python examples/text_processing/multilingual_translator.py
```

### Universal Composition

O script `universal_composition.py` demonstra:

- Composição avançada de processadores de texto
- Integração com outros componentes do framework
- Criação de pipelines complexos de processamento

Para executar:

```bash
python examples/text_processing/universal_composition.py
```

## Requisitos

Todos os exemplos requerem que o framework PepperPy esteja instalado. Você pode instalá-lo usando:

```bash
pip install -e ../..
```

ou

```bash
poetry install
```

a partir do diretório raiz do repositório.
