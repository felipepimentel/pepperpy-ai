# PepperPy Workflow Automation Examples

Este diretório contém exemplos demonstrando como usar as funcionalidades de automação de fluxos de trabalho do framework PepperPy.

## Exemplos Disponíveis

### Simple Intent

O script `simple_intent.py` demonstra:

- Reconhecimento básico de intenções
- Mapeamento de intenções para ações
- Processamento de comandos simples

Para executar:

```bash
python examples/workflow_automation/simple_intent.py
```

### Complex Workflow

O script `complex_workflow.py` demonstra:

- Criação de fluxos de trabalho complexos
- Encadeamento de múltiplas etapas
- Tratamento de condições e ramificações

Para executar:

```bash
python examples/workflow_automation/complex_workflow.py
```

### Custom Components

O script `custom_components.py` demonstra:

- Criação de componentes personalizados
- Extensão do framework com novas funcionalidades
- Integração de componentes personalizados em fluxos

Para executar:

```bash
python examples/workflow_automation/custom_components.py
```

### Parallel Pipeline Example

O script `parallel_pipeline_example.py` demonstra:

- Execução paralela de etapas de processamento
- Otimização de desempenho com paralelismo
- Sincronização de resultados paralelos

Para executar:

```bash
python examples/workflow_automation/parallel_pipeline_example.py
```

### Parallel Processing

O script `parallel_processing.py` demonstra:

- Processamento paralelo de dados
- Distribuição de carga de trabalho
- Agregação de resultados de processamento paralelo

Para executar:

```bash
python examples/workflow_automation/parallel_processing.py
```

### Workflow Orchestration

O script `workflow_orchestration.py` demonstra:

- Orquestração de múltiplos fluxos de trabalho
- Gerenciamento de dependências entre fluxos
- Monitoramento e controle de execução

Para executar:

```bash
python examples/workflow_automation/workflow_orchestration.py
```

## Requisitos

Todos os exemplos requerem que o framework PepperPy esteja instalado. Você pode instalá-lo usando:

```bash
pip install -e ../..
```

ou

```bash
poetry install
```

a partir do diretório raiz do repositório.
