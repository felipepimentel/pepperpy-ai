# PepperPy Assistants Examples

Este diretório contém exemplos demonstrando como usar as funcionalidades de assistentes do framework PepperPy.

## Exemplos Disponíveis

### Research Assistant

O script `research_assistant.py` demonstra:

- Criação de um assistente de pesquisa
- Processamento de consultas de pesquisa
- Geração de respostas baseadas em conhecimento

Para executar:

```bash
python examples/assistants/research_assistant.py
```

### Virtual Assistant

O script `virtual_assistant_example.py` demonstra:

- Implementação de um assistente virtual
- Gerenciamento de contexto de conversação
- Processamento de comandos e consultas

Para executar:

```bash
python examples/assistants/virtual_assistant_example.py
```

## Requisitos

Todos os exemplos requerem que o framework PepperPy esteja instalado. Você pode instalá-lo usando:

```bash
pip install -e ../..
```

ou

```bash
poetry install
```

a partir do diretório raiz do repositório.
