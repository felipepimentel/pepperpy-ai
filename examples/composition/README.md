# PepperPy Composition Examples

Este diretório contém exemplos demonstrando como usar as funcionalidades de composição do framework PepperPy.

## Exemplos Disponíveis

### Basic Composition Demo

O script `basic_composition_demo.py` demonstra:

- Composição básica de componentes
- Criação de pipelines de processamento
- Transformação de dados através de composição

Para executar:

```bash
python examples/composition/basic_composition_demo.py
```

### Pipeline Builder Demo

O script `pipeline_builder_demo.py` demonstra:

- Uso avançado do construtor de pipelines
- Composição de múltiplos processadores
- Bifurcação e junção de fluxos de dados

Para executar:

```bash
python examples/composition/pipeline_builder_demo.py
```

## Requisitos

Todos os exemplos requerem que o framework PepperPy esteja instalado. Você pode instalá-lo usando:

```bash
pip install -e ../..
```

ou

```bash
poetry install
```

a partir do diretório raiz do repositório.
