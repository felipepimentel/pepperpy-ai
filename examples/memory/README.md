# PepperPy Memory Examples

Este diretório contém exemplos demonstrando como usar as funcionalidades de gerenciamento de memória do framework PepperPy.

## Exemplos Disponíveis

### Simple Memory

O script `simple_memory.py` demonstra:

- Armazenamento básico de memórias de conversas
- Recuperação de histórico de conversas
- Gerenciamento de múltiplas conversas

Para executar:

```bash
python examples/memory/simple_memory.py
```

### Memory Example

O script `memory_example.py` demonstra:

- Recursos avançados de gerenciamento de memória
- Priorização e filtragem de memórias
- Integração com outros componentes do framework

Para executar:

```bash
python examples/memory/memory_example.py
```

## Requisitos

Todos os exemplos requerem que o framework PepperPy esteja instalado. Você pode instalá-lo usando:

```bash
pip install -e ../..
```

ou

```bash
poetry install
```

a partir do diretório raiz do repositório.
