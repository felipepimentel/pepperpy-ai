# PepperPy Content Generation Examples

Este diretório contém exemplos demonstrando como usar as funcionalidades de geração de conteúdo do framework PepperPy.

## Exemplos Disponíveis

### Article Generator

O script `article_generator.py` demonstra:

- Geração de artigos completos
- Estruturação de conteúdo
- Personalização de estilo e tom

Para executar:

```bash
python examples/content_generation/article_generator.py
```

### News Podcast Generator

O script `news_podcast_generator.py` demonstra:

- Geração de scripts para podcasts
- Transformação de notícias em conteúdo de áudio
- Formatação específica para mídia falada

Para executar:

```bash
python examples/content_generation/news_podcast_generator.py
```

## Requisitos

Todos os exemplos requerem que o framework PepperPy esteja instalado. Você pode instalá-lo usando:

```bash
pip install -e ../..
```

ou

```bash
poetry install
```

a partir do diretório raiz do repositório.
