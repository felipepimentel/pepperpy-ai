# PepperPy RAG Examples

Este diretório contém exemplos demonstrando como usar as funcionalidades de Retrieval Augmented Generation (RAG) do framework PepperPy.

## Exemplos Disponíveis

### Document QA

O script `document_qa.py` demonstra:

- Processamento de documentos para RAG
- Criação de um sistema de perguntas e respostas baseado em documentos
- Recuperação de informações relevantes para responder consultas

Para executar:

```bash
python examples/rag/document_qa.py
```

## Recursos

O diretório `documents/` contém documentos de exemplo que são usados pelos exemplos de RAG.

## Requisitos

Todos os exemplos requerem que o framework PepperPy esteja instalado. Você pode instalá-lo usando:

```bash
pip install -e ../..
```

ou

```bash
poetry install
```

a partir do diretório raiz do repositório.
